# PKM-KC: **Tunas Karsa**

## Pendahuluan

### Latar Belakang
Kemajuan teknologi membawa dampak signifikan terhadap perilaku anak-anak. Permainan digital telah menjadi aktivitas utama yang dilakukan tanpa batas waktu, mengakibatkan kebiasaan negatif, seperti menurunnya prestasi akademik. Oleh karena itu, diperlukan solusi inovatif untuk mengatasi kecanduan game pada anak.

### Tujuan
- Mengembangkan aplikasi edukasi berbasis sistem **reward and punishment**.
- Membantu anak membatasi waktu bermain game.
- Meningkatkan kesadaran pentingnya pendidikan melalui gamifikasi.
- Memfasilitasi kontrol orang tua terhadap aktivitas anak.

### Prediksi Manfaat
- Membantu anak mengatur waktu antara bermain dan belajar.
- Mengurangi ketergantungan anak terhadap game secara edukatif.
- Meningkatkan hasil belajar melalui penggabungan elemen edukasi dalam permainan.

### Luaran
- Laporan kemajuan.
- Laporan akhir.
- Akun media sosial.
- Prototipe aplikasi dengan anggaran Rp 8.000.000.

### Rumusan Masalah
1. Bagaimana cara membatasi waktu bermain anak tanpa merusak hubungan dengan orang tua?
2. Bagaimana meningkatkan fokus anak pada akademik?
3. Bagaimana membuat anak belajar secara mandiri?

## Tinjauan Pustaka

### Kecanduan Game pada Anak
- **Definisi WHO:** Gangguan perilaku akibat kecanduan game.
- **Tanda-tanda:** Ketergantungan, prioritas bermain, mengabaikan konsekuensi negatif.

### Dampak Kecanduan Game terhadap Pendidikan Anak
- Penurunan prestasi akademik.
- Kurangnya fokus dan konsentrasi.
- Pengurangan waktu untuk aktivitas edukatif.
- Risiko kesehatan mental, seperti stres dan isolasi sosial.

### Aplikasi Edukasi sebagai Solusi
Pendekatan game-based learning dapat menyeimbangkan kebutuhan belajar dan bermain anak. Contoh aplikasi: **Kahoot**, **Quizizz**, **Duolingo**.
